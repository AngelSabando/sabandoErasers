# sabandoErasers

Exercise
51-100. Design and implement one dynamic web page to process information from a database table/collection for the following problem: A company X has a system currently running, among other data, its system stores information about (The topic is according to your number in the roster(list)). You must think in at least 8 attributes for your element plus an ID field. A web client and a PHP page that uses ORM technology are needed to insert/find/read all the information into from/to the database (MongoDB Atlas/MariaDB or any other you have access IN THE CLOUD). Your web page must be uploaded to the cloud to have full points in the exercise. Also, you have to compute some value based on the fields, the computation must be done using either JavaScript/TypeScript or PHP.
